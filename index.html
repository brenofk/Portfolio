Portfólio

Projeto Acadêmico – Frontend

Projeto acadêmico desenvolvido durante a graduação, inicialmente focado na refatoração do frontend de um site de música. O frontend foi desenvolvido com HTML e CSS, e foram adicionadas funcionalidades interativas utilizando JavaScript, que não estavam presentes no código original, como ações de clique em botões, reprodução de músicas no próprio site e cards animados de artistas.

Projeto Acadêmico – Backend

Em um segundo momento, foi desenvolvido o backend da aplicação utilizando Python, linguagem definida na disciplina. O sistema passou a contar com cadastro e autenticação de usuários utilizando o framework Django, além de funcionalidades completas de CRUD para playlists e músicas. Os dados foram persistidos em banco de dados MySQL.

Projeto de TCC – Aplicativo Mobile para Gerenciamento de Pets

Aplicativo mobile desenvolvido como Trabalho de Conclusão de Curso, com foco em dispositivos móveis, voltado ao gerenciamento de vacinas e medicamentos de animais domésticos. A aplicação permite cadastro e autenticação de usuários, gerenciamento de pets e controle completo de vacinas associadas a cada animal.

O sistema oferece funcionalidades como criação de conta, login, cadastro de pets e associação de vacinas por animal, com interface projetada a partir de princípios de usabilidade para facilitar a experiência do usuário.

  • Autenticação de usuários
  • CRUD completo de usuários, pets e vacinas
  • Associação de vacinas por pet
  • Foco em usabilidade e experiência do usuário

O projeto contou com levantamento de requisitos funcionais e não funcionais, diagramas de caso de uso e documentação técnica completa.

Como resultado, o sistema funcionou corretamente em ambiente web. No ambiente mobile (Expo Go), houve uma limitação relacionada à configuração de rede, documentada como ponto de melhoria futura.
